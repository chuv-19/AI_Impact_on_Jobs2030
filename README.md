# Анализ базы данных «AI Impact on Jobs 2030»
**Цель работы:** На основе статистических данных выявить зависимости между уровнями образования, заработной платой, профессиональными навыками специалистов и вероятностью автоматизации их рабочих мест к 2030 году.

**Основные задачи проекта:**
* Выполнение загрузки, очистки и предварительной обработки данных (коррекция типов данных, обработка пропущенных значений и дубликатов);
* Проведение детального разведочного анализа (EDA) и расчет расширенных статистических характеристик;
* Исследование распределения рисков автоматизации по категориям профессий;
* Построение матриц корреляции для определения наиболее защищенных групп навыков;
* Формулирование стратегических выводов для соискателей, образовательных учреждений и бизнеса.

## 2. Исходные данные
Объектом исследования является набор данных `AI_Impact_on_Jobs_2030.csv` (3000 наблюдений), содержащий следующие ключевые параметры:
* `Job_Title` — наименование должности;
* `Average_Salary` — среднегодовая заработная плата;
* `Years_Experience` — стаж (опыт работы) специалиста;
* `Education_Level` — базовый уровень образования;
* `AI_Exposure_Index` — индекс интенсивности внедрения ИИ в текущие рабочие процессы;
* `Tech_Growth_Factor` — технологический фактор роста отрасли;
* `Automation_Probability_2030` — расчетная вероятность полной автоматизации профессии к 2030 году;
* `Risk_Category` — уровень угрозы замещения (`Low`, `Medium`, `High`);
* `Skill_1` — `Skill_10` — нормированные индексы значимости различных групп профессиональных и гибких (soft) навыков.

## 3. Стек технологий
Проект реализован с использованием современного аналитического стека библиотек языка Python 3:
* **Pandas** — считывание, очистка, фильтрация и агрегация данных;
* **NumPy** — проведение векторных вычислений и математических операций;
* **Matplotlib** — проектирование графиков и базовых визуализаций;
* **Seaborn** — построение матриц корреляции и продвинутых статистических диаграмм;
* **Jupyter Notebook** — интерактивная среда разработки кода и фиксации результатов.
